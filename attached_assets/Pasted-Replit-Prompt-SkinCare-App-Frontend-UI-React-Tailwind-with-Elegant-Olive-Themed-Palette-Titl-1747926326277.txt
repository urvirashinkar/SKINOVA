📌 Objective:
Build a responsive, aesthetic frontend for a skin analysis app using React + Tailwind CSS. The UI should feel clean, natural, and dermatology-aligned, using the following calming, skin-friendly color palette.
🖌️ 🎨 Color Palette (Tailwind Customization Required):
Purpose	Hex Code	Tailwind Class (Extend)
Primary	#C5C27E	bg-primary / text-primary
Secondary (Accent)	#8D8A5A	bg-olive / text-olive
Background	#FFFFFF	bg-white
Text Primary	#333333	text-gray-900
CTA Buttons (Success)	#6BBF59	bg-success / text-success
Error Alerts	#FF6B6B	bg-error / text-error
In tailwind.config.js, extend your theme:
theme: {
  extend: {
    colors: {
      primary: '#C5C27E',
      olive: '#8D8A5A',
      success: '#6BBF59',
      error: '#FF6B6B'
    }
  }
}
